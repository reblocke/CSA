# ATS 2020 Abstract Summary

## Citation

Sellman J, Locke BW, McFarland J, Uribe F, Sundar K. Spectrum of Central Sleep Apnea at an Academic Sleep Center Between 2016-2018 and Treatment Responses to CPAP vs. Adaptive Servo Ventilation. American Journal of Respiratory and Critical Care Medicine. 2020;201(Supplement_1):A2430. doi:10.1164/ajrccm-conference.2020.201.1_MeetingAbstracts.A2430.

Publisher record: https://doi.org/10.1164/ajrccm-conference.2020.201.1_MeetingAbstracts.A2430

## Repository Relationship

This repository preserves the legacy Python workflow for the single-center central sleep apnea cohort described in the ATS 2020 abstract. The code supports historical cohort description, treatment categorization, and local output generation when run against an authorized compatible workbook.

The repository does not include raw patient-level sleep-center or EHR data. It also does not mirror the full publisher abstract text or any publisher-formatted files.

## Project Summary

The abstract characterized adults with central sleep apnea codes at the University of Utah Sleep Medicine Program between 2016 and 2018 and summarized diagnostic patterns and treatment-response workflows involving CPAP and adaptive servo ventilation. The public repository is intended to make the historical Python workflow discoverable and inspectable while keeping restricted clinical data outside version control.

## Downstream Work

This repository is separate from `reblocke/CSA-CPAP-Prescribing`, which supports a later Stata-based paper workflow for CPAP prescribing and course after CPAP in patients with central sleep apneas.
